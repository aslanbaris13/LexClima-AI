## Overview

LexClima AI is an autonomous Climate-Tech platform that converts corporate sustainability reports into measurable carbon reduction action plans — detecting greenwashing, scoring climate compliance risk, and generating facility-specific roadmaps in **45 seconds** instead of the industry-standard 3 weeks.

> 🏁 **Ideathon project** — Developed and presented at the YZA (Yapay Zeka ve Teknoloji Akademisi) Ideathon, 2025. 4-person team.

---

## Problem Statement

53% of companies cannot back their green claims with concrete carbon data. Manual compliance analysis takes an average of 3 weeks, slowing green transition and increasing carbon emissions. Non-compliance with climate regulations risks fines of millions of euros — and raises the global 1.5°C threshold risk.

---

## How It Works

```
Step 0 — Trigger
  Sustainability report uploaded as PDF

Step 1 — Scout Agent (Watchdog)
  Scans global climate standards (COP decisions) 24/7
  Autonomously updates its knowledge base

Step 2 — Parser Agent (Segmentor)
  Splits document into semantic blocks via NLP
  Hunts empty "green" claims (greenwashing risks)

Step 3 — Auditor Agent (Inspector)
  Cross-checks company claims against global standards
  Generates climate compliance risk score

Step 4 — Advisor Agent (Consultant)
  Produces optimal carbon reduction action
  "Take this step at this facility, reduce X tons"

Step 5 — Output (45 seconds total)
  C-Level dashboard: real-time alerts
  Color-coded risk map
  Audit-ready compliance report
```

## System Architecture

| Layer | Technology |
|---|---|
| AI Layer | LangChain multi-agent orchestration + RAG (Retrieval-Augmented Generation) |
| LLM | On-premise Llama 3.1 (zero data leakage) |
| Data Pipeline | Global climate standards APIs + internal document integration |
| Backend | Python — real-time memory-based processing |
| Frontend | SaaS dashboard for managers and compliance teams |
